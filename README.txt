1/ Décompressez le fichier Projet.zip à la racine du disque C. (ou déplacer le dossier projet à la racine)

2/ Le programme nécessite l'installation de postgresql 9.5.
Avec pgadminIII, créez une base de données "images".

Cliquez sur la bdd images créée, puis sur l'onglet SQL dans la bar de menu.
Ouvrir le fichier "1-dbroles.sql" puis exécuter. Faites de même avec "2-dbdatas.sql".

3/ Eclipse nécessite tomcat v6 et jre 1.8.0_25
